Welcome to Smoothie

Congratulations, you have succesfully setup your Smoothieboard's network interface !

Compatible software

Now, software can talk to your Smoothieboard over the network.

Here is a list of software that you can use with this interface :

  • Pronterface
  • Visicut
  • Laserweb
  • Fabrica

On-board web interfaces

You can also install web interfaces created by the community to your SD card

You do so by downloading the interfaces from the internet, copying them to your SD card, then accessing them via the browser

Installation instructions can be found at the following address : smoothieware.org/install-web-interface

Note : If you were expecting to find Smoothie's "classic" web interface but found this instead, this is because recent Smoothie firmwares do not integrate the interface anymore and require you install it to the SD card instead. The interface took a lot of space in flash that is better used by code, and installing it on the SD card makes much more sense.
